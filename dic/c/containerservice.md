# Container Service

Linux Container Serviceから出発した環境仮想化技術の一つ。
現在はDocker、Kubernetesなどのサービスとして知られていて知らないインフラ担当はいない程有名な機能の一つになりました。

一般的な仮想サーバ(ハイパーバイザ型)と比較して、OSのカーネル部分などをコンテナ間で共通利用することで
アプリケーションなどの部分だけを仮想化することで、高速起動・軽量を実現しています。
